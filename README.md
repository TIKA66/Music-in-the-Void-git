# Music-in-the-Void-git

## Music in the Void
A 2D, rhythm-inspired endless runner set in a mysterious cosmic void.

## About the Game
Music in the Void started as an idea: *what if the obstacles in an endless runner weren't random, but pieces of entire worlds that had been erased?*
Start by choosing your vibe of music and set your pace as you venture deeper, strange fragments begin to emerge from the darkness: shattered relics, forgotten worlds, and echoes of places that should no longer exist. Each obstacle is a remnant of something lost, drifting through the abyss. The longer you survive, the more the void reveals its secret. It is not empty. It is feeding. Consuming entire realities and leaving behind only scattered traces of what once was. How long can you endure before you, too, become a forgotten fragment in the dark?

Your goal is simple:
**Survive for as long as possible.**

## Controls
- **Space Bar** → Jump
- **Down Arrow** → Duck

## Features

* Endless survival gameplay
* Rhythm-inspired obstacle spawning
* Increasing difficulty over time
* Atmospheric space-themed visuals
* Lore revealed through the obstacles themselves
* Original soundtrack and sound effects

### Built With

* Godot Engine
* GDScript

## Why I Made This
I wanted to create a game that combined fast-paced gameplay with environmental storytelling. Instead of explaining everything through dialogue or cutscenes, the story is revealed through the objects you encounter while playing.

This project has also been a great way for me to improve my skills in:
* Game development
* Programming with GDScript
* Audio integration
* Scene management
* UI design
* Game balancing
